Projets realises au cours de la formation

vers une expertise des technologies modernes

JEE : Gestion d'une pharmacie en JEE ou equivalent

Chaque étudiant sera ramener à créer une application web en mettant en pratique ses connaissances basiques de technologie JEE apprises en classe. Proposition de solution

uml2.0 : Conception d'une application de Gestion de conferences

Chaque étudiant sera invité à mettre en pratique ses compétences de conception UML2.0 en conçevant toute une application de gestion de conférences: comités, membres de comités, auteurs, conférenciers ... etc. NB: Une implémentation de ladite application est demandée et ce projet est réalisé en binôme. Proposition de solution

python : Segmentation de texte francais ou arabe

Avec le langage python, le langage naturel est devenu facilement segmentable. Sur ce projet, les étudiants seront challengés à réaliser une application plus ou moins complexe, permettant la segmentation de texte français ou arabe NB: De même, une application de gestion d'une banque sera réalisée avec le même module.

EJB ‐ SPRING : Archivage ou application e-learning

Pour mettre en évidence leurs connaissances en java EE avancée, les étudiants sont emmenés à réaliser en deux groupe, une application robuste d'archivage de données et concevoir une plateforme d'e ‐ learning implementée soit avec la spécification de la java EE : EJB, soit avec le fameux framework java EE : SPRING NB: De même, une application de gestion de bons de cadeaux doit être implementée en examen.

Architecture orientee MDA : Realsiation d'un editeur de diagramme

L’architecture dirigée par les modèles ou MDA (pour l'anglais model driven architecture) est une démarche de réalisation de logiciels. C'est une variante particulière de l'ingénierie dirigée par les modèles (IDM, ou MDE pour l'anglais model driven engineering). Par groupes, les étudiants doivent développer un logiciel d'édition des diagrammes UML.

Securite de donnes : E-Voting securise

En binômes ou trinômes, les étudiants sont emmenes à similer une application securisée de vote en ligne en utilisant les méthodes modernes de cryptages de données telles que: la technologie blockchain

XML & Web Services : Divers sujets

En XML & Web Services, divers sujets sont donnés à des groupes d'étudiants. Des sujets d' implémentation et d'utilisations des service Web: Temperature ville, Conversion de devises, pharmacie de garde, ... etc

DB NoSQL : Divers sujets concernant les DB NoSQL

Face à un monde de données, les bénéficiants de la formation seront emmenés à travailler avec de données massives dans divers entreprises. C'est pourquoi il est indispensable qu'ils se familiarisent et de l'environement de Big Data et des bases de données NoSQL. Des projets comme Neo4j, ... etc leurs sont afféctés.

Business Intelligence : Societe Orion : ETL Talend Studio

Cette société fictive, présente au niveau mondial, est spécialisée dans la commercialisation d’articles de sport et d’extérieur. Les données disponibles regroupent des informations sur : les employés, les produits, les clients, les commandes, les fournisseurs. Les etudiants vont ainsi utilisé les outils d'informatique décisionnelle visant plusieurs axes dans l'objectif d'aider la société orion à en prendre de bonnes décisions. Proposition de solution

Big Data environement : Composants de l'environement

Chaque technologie de Big Data environement est affecté à un étudiant. L'objectif étant d'acquérir d'amples connaissances sur ladite technologie, de la comprendre dans les détails possibles puis de l'exécuter pour enfin l'intégrer dans Apache Hadoop

Data Visualisation : Visualisation, Machine Learning et Deep Learning

Ici, ceux qui bénéficient la formation sont laissés au libre choix de travailler sur n'importe quel dataset qui leur convient. Chaque projet doit utiliser les outils de Visualisation enseignés par le prof durant la formation, les algorithmes de deep learning mais pas que. Proposition de solution
